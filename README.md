# :information_source: Como usar

```bash
# Clone este repositório:
$ git clone https://github.com/JonataRamalho/pwa-news.git

# Vá para o repositório:
$ cd pwa-news

# Instale as dependências:
$ yarn install

# Inicie o React:
$ yarn start
```
